注意本项目为保证隐私没有上传配置文件

在conf文件夹下面新建config.yaml文件

可以自行按照下面格式配置
```azure

system:
  env: 
  db-type: 
  oss-type: 
  router-prefix: 
  addr: 
  chat_addr: 
  iplimit-count: 
  iplimit-time: 
  use-multipoint: 
  use-redis: 
  use-mongo: 

mysql_user:
  username: 
  password: 
  path: 
  port: 
  db_name: 

mongo:
  username: 
  password: 
  path: 
  port: 
  db_name: 

```
